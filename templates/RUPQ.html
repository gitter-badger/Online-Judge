Problem : Updates and Queries | Time limit: 0.03secs | Memory: 2MB | Submit code

Given three positive integers 'n','u','s' The next 'u' lines of the input contains two integers 'a' and 'b' seperated by a single space. Coming to explanation of the prolem, assume an array of size 'n' in which each of its elements is initialised to 1.
For each 'a' and 'b' given, increment the elements of the array from indexed 'a' to 'b' inclusively.
Finally after 'u' updates output the sum of the array till the index 's'(inclusively).

Constraints:
n<=100 , u<=10, s<=br <99>
Input:
First line of input contains three positive integers 'n','u','s'.
Each of the 'u' lines contain two single space seperated integers 'a' and 'b'.


Output:
After 'u' operations, output the sum of the array till the index 's' (inclusively).

Sample Input:


10 3 7
1 3
2 6
5 8

Sample Output:
19
