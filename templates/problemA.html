Problem : The easiest problem | Time limit: 0.03secs | Memory: 2MB | Submit code

Given two positive integers a and b each of which is greater than 10^20.

Your task is to find the sum of the given two integers. Simple. Isn't it?

Constraints:
a>10^20 and b>10^20
Number of testcases t <= 10.

Input:
First line of input contains integer 't' representing the number of test cases.
Each test case contains two integers seperated by a single space.

Output:
For each test case print the sum of the two integers in new line.

Sample Input:

3
100000000000000000000000 200000000000000000000000
321547895412358741242 514261470231054879654
111111111111111111111111111111 222222222222222222222222222222

Sample Output:
30000000000000000000000
835709276643303621996
333333333333333333333333333333
