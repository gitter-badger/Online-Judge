Problem : Character Count | Time limit: 0.02secs | Memory: 500B | Submit code

A straight forward problem again!

Given a string S and few characters, Output the given character and count of the character in the string in the ascending order of their count. If two characters have same count, then print them in order of their occurence in alphabets.

Constraints:
1<=length|S|<=10 power 20.
1<=n<=2000000000.
'S' consists of only lower case alphabets.

Input:
Each test case consists of three lines.
Line1 : String S
Line2 : 'n', number of characters
Line3 : n characters seperated by spaces.

Output:
Output n lines, each line containing the character and it's count seperated by a single space. Don't forget to print the output in the sorted order of the count of characters
If count of two characters are same, print them in the lexical order.

Sample Input:

science

Sample Output:
i 1
n 1
s 1
c 2
e 2
