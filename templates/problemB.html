Problem : Mysterious Library | Time limit: 0.02secs | Memory: 1.5MB | Submit code

Vinay wants to learn Algorithms. So he went to the library of MVSR and was confused on seeing the books.
Suddenly the librarian came and advised vinay that in each rack the m-th and n-th books from both sides of the rack are same.

Your task is to find the total number of books in each rack.

Constraints:
1<=m,n<=2000000000.
Number of testcases t <= 10.

Input:
First line of input contains integer 't' representing the number of test cases.
Each of the next t lines has two numbers n and m representing the n-th book and m-th book which are same from both the sides of the rack.

Output:
For each test case output total number of books in the rack.

Sample Input:

1
2 3


Sample Output:

4
