Problem : The Dynamic phobia | Time limit: 0.04secs | Memory: 800B | Submit code

As we know that Vinay went to the library to learn Algorithms, he took a book of String theory and started reading it.
The book confused him so much that he got a phobia towards palindromes. He used to act in a weird way on seeing a palindrome.
To cure him of this fatal disease, doctors from all over Hyderabad discussed his fear and decided to expose him to large number of palindromes.

To do this, they decided to play a game with Vinay. The rules of the game are as follows:

Vinay has to supply a string S. The doctors have to add or insert characters to the string to make it a palindrome. Characters can be inserted anywhere in the string.
The doctors took this game very lightly and just appended the reverse of S to the end of S, thus making it a palindrome.
For example, if S = "fft", the doctors change the string to "ffttff" and if S = "MVSR", doctors change the string to "MVSRRSVM". Nowadays, Vinay is cured of the disease (having been exposed to a large number of palindromes), but he still wants to continue the game by his rules. He now asks the doctors to insert the minimum number of characters needed to make S a palindrome.

Help the doctors accomplish this task.
For instance, if S = "fft", the doctors should change the string to "tfft", adding only 1 character.


Input:
Each test case consists of one line, the string S. The length of S will be no more than 6100 characters, and S will contain no whitespace characters.

Output:
For each test case output one line containing a single integer denoting the minimum number of characters that must be inserted into S to make it a palindrome.

Sample Input:

fft
MVSR

Sample Output:
1
3



Explanation:

In the first saample testcase, as S = "fft", the doctors should change the string to "tfft", adding only 1 character. But in the second test case, we need to add atleast 3 characters to make it a palindrome.
